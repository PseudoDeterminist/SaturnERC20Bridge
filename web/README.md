# Saturn Wrapping Suite UI

Static vanilla JS UI for minting/redeeming the Saturn bridge stack.

## Run
Open index.html in a browser or serve the folder with any static server.

## Notes
- Requires MetaMask connected to ETC (chainId 61).
- Mint STRN uses ERC223 transfer with the MINT tag.
- Mint STRN10K uses STRN approvals and exact 10,000.0000 lots.
